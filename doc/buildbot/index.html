Weekly distribution builds

Automatically produced from our SVN repository every week, these builds have passed our unit tests but have not been checked by a human on the relevant platform. To download an official release of Topographica, please see topographica.org/Downloads.

  • Platform-independent tar.gz source archives (for "python setup.py install" into existing python environment e.g. EPD 7)
  • Windows exe (for installation into existing python environment e.g. EPD 7)
  • Linux source DEBs or source RPMs
