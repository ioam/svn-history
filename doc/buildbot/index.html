Weekly distribution builds

Automatically produced from our SVN repository every week, these builds have passed our unit tests but have not been checked by a human. To download an official release of Topographica, please see topographica.org/Downloads.

  • Linux DEBs or RPMs
  • Windows exe or msi (for installation into existing python environment)
  • Platform-independent tar.gz source archives (for "python setup.py install" into existing python environment)
