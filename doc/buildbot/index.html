Topographica: automatic tests and builds

These tests and builds are run automatically. You can use the test results to see the status of the latest version of Topographica on your platform. The automatic builds allow you to get the latest versions of Topographica without using SVN, but note that they have probably not been tested by a human. To download an official release of Topographica, please see topographica.org/Downloads.
